What is LearnSpire?

Project learnSpire was initiated with the purpose of providing best in class tools for learning management by t8 inc. We aspire to deliver great technologies that will help students as well as educators to better manage their learning and teaching activities while being incredibly reliable. For better software administration, the entire project is separated into five sub projects.

  • 1. User Management System: Scalable REST API for user management and authentication
  • 2. Learning Management System: REST API developed to record and manage learner activities
  • 3. learnSpire Mobile Application: Android application for users to access REST APIs
  • 4. learnSpire Web Application: Web application for users to access REST APIs
  • 5. learnSpire Official Website: To gain insights about project overview, structure, and architecture

How we improved our software

following are the distinguishing features of our software and development process.

  • DevOps
  • Security
  • Performance
  • Testing
  • Containerization: Both Backend applications are hosted as Docker containers in cloud virtual machines.
  • Database: We hosted the database in a cloud virtual machine.
  • Deployment Process: We designed a little CI/CD pipeline to make our deployment process fast and easy.
  • Domain: A domain name is added to make the application easier to use.
  • Caching: We added spring in-memory caching and increased the performance by 40%.
  • Multi-Threading: We added multi-threading to the email service and increased the performance by 70%.
  • SSL Certification: We added a Certbot free SSL certificate to secure the connection.
  • Hiding configuration details: We didnt expose our secrets and key in the application properties for security reasons. Instead we provided them as environmental variables.
  • Unit Testing: All of the backend services are unit tested to ensure that nothing goes wrong in any situation.
  • Integration Testing: All the endpoints are integration tested to ensure that all functionalities work as intended.
